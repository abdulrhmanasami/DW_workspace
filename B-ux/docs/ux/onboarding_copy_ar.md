# Onboarding Copy — Arabic (AR)

**Version**: 1.0.0  
**Created by**: Cursor B-ux  
**Last updated**: 2025-11-25  
**Purpose**: محتوى الترجمة للتطبيق الأولي والإرشادات داخل التطبيق (متوافق مع Sale-Only)

---

## الإرشادات

1. **توافق Sale-Only**: جميع النصوص يجب أن تكون صادقة حول توفر الميزات.
2. **لغة مشروطة**: استخدام عبارات مثل "عند التوفر" للميزات المعتمدة على الخادم.
3. **لا وعود كاذبة**: لا تعد أبداً بميزات غير منفذة أو غير مفعّلة.
4. **واضح ومختصر**: اجعل النص قصيراً وقابلاً للتنفيذ.

## تدفق التطبيق الأولي — العميل

### الخطوة 1: الترحيب

| Key | Value |
|-----|-------|
| `onb_welcome_title` | أهلاً بك في ديليفري ويز |
| `onb_welcome_body` | شريكك الموثوق للتوصيل. اطلب ما تحتاجه وتتبع توصيلتك بالوقت الفعلي. |

### الخطوة 2: مقدمة التطبيق

| Key | Value |
|-----|-------|
| `onb_app_intro_title` | كيف يعمل |
| `onb_app_intro_body` | تصفح المنتجات، قدم طلبك، ونوصله لباب بيتك. بسيط وسريع. |

### الخطوة 3: الطلب

| Key | Value |
|-----|-------|
| `onb_ordering_title` | طلب سهل |
| `onb_ordering_body` | اعثر على ما تحتاجه، أضفه للسلة، وادفع في ثوانٍ. خيارات دفع متعددة متوفرة حيث يُدعم ذلك. |

### الخطوة 4: التتبع (مشروط)

| Key | Value |
|-----|-------|
| `onb_tracking_title` | تتبع طلبك |
| `onb_tracking_body` | تابع توصيلتك بالوقت الفعلي عندما يكون التتبع متاحاً في منطقتك. ستتلقى تحديثات في كل خطوة. |

### الخطوة 5: الأمان والخصوصية

| Key | Value |
|-----|-------|
| `onb_security_title` | أمانك يهمنا |
| `onb_security_body` | بياناتك محمية بمعايير أمان عالمية. لن نشارك معلوماتك الشخصية أبداً بدون موافقتك. |

### الخطوة 6: الإشعارات (إذن)

| Key | Value |
|-----|-------|
| `onb_notifications_title` | ابق على اطلاع |
| `onb_notifications_body` | فعّل الإشعارات لتلقي تحديثات الطلبات وتنبيهات التوصيل والعروض الحصرية. |

### الخطوة 7: جاهز

| Key | Value |
|-----|-------|
| `onb_ready_title` | أنت جاهز! |
| `onb_ready_body` | ابدأ الاستكشاف وقدم طلبك الأول. نحن هنا لمساعدتك متى احتجت. |

## تدفق التطبيق الأولي — السائق

### الخطوة 1: الترحيب

| Key | Value |
|-----|-------|
| `onb_rider_welcome_title` | أهلاً بك أيها السائق! |
| `onb_rider_welcome_body` | انضم لشبكة التوصيل وابدأ بالكسب. ساعات مرنة وأجر عادل. |

### الخطوة 2: كيف يعمل

| Key | Value |
|-----|-------|
| `onb_rider_how_it_works_title` | رحلتك تبدأ هنا |
| `onb_rider_how_it_works_body` | اقبل التوصيلات، توجه للاستلام، وصّل للعملاء. تتبع أرباحك في التطبيق. |

### الخطوة 3: الموقع

| Key | Value |
|-----|-------|
| `onb_rider_location_title` | تفعيل الموقع |
| `onb_rider_location_body` | نستخدم موقعك لربطك بالتوصيلات القريبة وتوفير التنقل. يُشارك موقعك فقط أثناء التوصيلات النشطة. |

### الخطوة 4: الأمان

| Key | Value |
|-----|-------|
| `onb_rider_security_title` | آمن ومحمي |
| `onb_rider_security_body` | أرباحك وبياناتك الشخصية محمية. المصادقة متعددة العوامل تحافظ على أمان حسابك. |

### الخطوة 5: الإشعارات

| Key | Value |
|-----|-------|
| `onb_rider_notifications_title` | لا تفوت أي توصيلة |
| `onb_rider_notifications_body` | احصل على تنبيهات فورية لطلبات التوصيل الجديدة والتحديثات المهمة. |

### الخطوة 6: جاهز

| Key | Value |
|-----|-------|
| `onb_rider_ready_title` | جاهز للتوصيل! |
| `onb_rider_ready_body` | أنت مُعدّ وجاهز للانطلاق. ابدأ بقبول التوصيلات الآن. |

| `hint_auth_biometric_body` | استخدم بصمتك أو وجهك لتسجيل الدخول بسرعة مع الحفاظ على أمان حسابك. |

| `hint_payments_limited_body` | بعض طرق الدفع قد لا تكون متوفرة في منطقتك. الدفع عند الاستلام متاح حيث يُدعم. |
