# Onboarding Copy — German (DE)

**Version**: 1.0.0  
**Created by**: Cursor B-ux  
**Last updated**: 2025-11-25  
**Purpose**: Lokalisierungsinhalte für Onboarding und In-App-Anleitungen (Sale-Only konform)

---

## Richtlinien

1. **Sale-Only Konformität**: Alle Texte müssen ehrlich über die Verfügbarkeit von Funktionen sein.
2. **Bedingte Sprache**: Verwenden Sie Ausdrücke wie "wenn verfügbar" für backend-abhängige Funktionen.
3. **Keine falschen Versprechen**: Niemals Funktionen versprechen, die nicht implementiert oder aktiviert sind.
4. **Klar & Prägnant**: Texte kurz und handlungsorientiert halten.

---

## Onboarding Flow — Kunde

### Schritt 1: Willkommen

| Key | Value |
|-----|-------|
| `onb_welcome_title` | Willkommen bei Delivery Ways |
| `onb_welcome_body` | Ihr zuverlässiger Lieferpartner. Bestellen Sie, was Sie brauchen, und verfolgen Sie Ihre Lieferung in Echtzeit. |

### Schritt 2: App-Einführung

| Key | Value |
|-----|-------|
| `onb_app_intro_title` | So funktioniert's |
| `onb_app_intro_body` | Produkte durchsuchen, bestellen und wir liefern an Ihre Tür. Einfach und schnell. |

### Schritt 3: Bestellung

| Key | Value |
|-----|-------|
| `onb_ordering_title` | Einfach Bestellen |
| `onb_ordering_body` | Finden Sie, was Sie brauchen, legen Sie es in den Warenkorb und bezahlen Sie in Sekunden. Mehrere Zahlungsoptionen wo unterstützt. |

### Schritt 4: Tracking (Bedingt)

| Key | Value |
|-----|-------|
| `onb_tracking_title` | Bestellung Verfolgen |
| `onb_tracking_body` | Verfolgen Sie Ihre Lieferung in Echtzeit, wenn Tracking in Ihrer Region verfügbar ist. Sie erhalten Updates bei jedem Schritt. |

### Schritt 5: Sicherheit & Datenschutz

| Key | Value |
|-----|-------|
| `onb_security_title` | Ihre Sicherheit ist wichtig |
| `onb_security_body` | Ihre Daten sind mit branchenüblicher Sicherheit geschützt. Wir geben Ihre persönlichen Daten niemals ohne Zustimmung weiter. |

### Schritt 6: Benachrichtigungen (Berechtigung)

| Key | Value |
|-----|-------|
| `onb_notifications_title` | Bleiben Sie informiert |
| `onb_notifications_body` | Aktivieren Sie Benachrichtigungen, um Bestellupdates, Lieferbenachrichtigungen und exklusive Angebote zu erhalten. |

### Schritt 7: Bereit

| Key | Value |
|-----|-------|
| `onb_ready_title` | Alles bereit! |
| `onb_ready_body` | Starten Sie und geben Sie Ihre erste Bestellung auf. Wir sind hier, um Ihnen zu helfen. |

## Onboarding Flow — Fahrer

### Schritt 1: Willkommen

| Key | Value |
|-----|-------|
| `onb_rider_welcome_title` | Willkommen, Fahrer! |
| `onb_rider_welcome_body` | Werden Sie Teil unseres Liefernetzwerks und verdienen Sie. Flexible Arbeitszeiten, faire Vergütung. |

### Schritt 2: So funktioniert's

| Key | Value |
|-----|-------|
| `onb_rider_how_it_works_title` | Ihre Reise beginnt hier |
| `onb_rider_how_it_works_body` | Lieferungen annehmen, zur Abholung navigieren, an Kunden liefern. Verfolgen Sie Ihre Einnahmen in der App. |

### Schritt 3: Standort

| Key | Value |
|-----|-------|
| `onb_rider_location_title` | Standort aktivieren |
| `onb_rider_location_body` | Wir verwenden Ihren Standort, um Sie mit Lieferungen in der Nähe zu verbinden. Ihr Standort wird nur während aktiver Lieferungen geteilt. |

### Schritt 4: Sicherheit

| Key | Value |
|-----|-------|
| `onb_rider_security_title` | Sicher & Geschützt |
| `onb_rider_security_body` | Ihre Einnahmen und persönlichen Daten sind geschützt. Multi-Faktor-Authentifizierung hält Ihr Konto sicher. |

### Schritt 5: Benachrichtigungen

| Key | Value |
|-----|-------|
| `onb_rider_notifications_title` | Keine Lieferung verpassen |
| `onb_rider_notifications_body` | Erhalten Sie sofortige Benachrichtigungen für neue Lieferanfragen und wichtige Updates. |

### Schritt 6: Bereit

| Key | Value |
|-----|-------|
| `onb_rider_ready_title` | Bereit zum Liefern! |
| `onb_rider_ready_body` | Sie sind eingerichtet und startklar. Beginnen Sie jetzt mit der Annahme von Lieferungen. |

## CTA Buttons

| Key | Value |
|-----|-------|
| `onb_cta_get_started` | Los geht's |
| `onb_cta_next` | Weiter |
| `onb_cta_skip` | Überspringen |
| `onb_cta_enable_notifications` | Benachrichtigungen aktivieren |
| `onb_cta_enable_location` | Standort aktivieren |
| `onb_cta_start_ordering` | Jetzt bestellen |
| `onb_cta_start_delivering` | Jetzt liefern |
| `onb_cta_maybe_later` | Vielleicht später |
| `onb_cta_done` | Fertig |
| `onb_cta_back` | Zurück |

## In-App Hints — Auth

| Key | Value |
|-----|-------|
| `hint_auth_phone_title` | Sichere Anmeldung |
| `hint_auth_phone_body` | Wir senden einen Bestätigungscode an diese Nummer. Ihre Telefonnummer hilft uns, Ihr Konto zu schützen. |
| `hint_auth_otp_title` | Prüfen Sie Ihre Nachrichten |
| `hint_auth_otp_body` | Geben Sie den Code ein, den wir an Ihr Telefon gesendet haben. Dies bestätigt, dass Sie es wirklich sind. |
| `hint_auth_2fa_title` | Zusätzlicher Schutz |
| `hint_auth_2fa_body` | Zwei-Faktor-Authentifizierung fügt eine zusätzliche Sicherheitsebene zu Ihrem Konto hinzu. |
| `hint_auth_biometric_title` | Schneller Zugriff |
| `hint_auth_biometric_body` | Verwenden Sie Ihren Fingerabdruck oder Ihr Gesicht, um sich schneller anzumelden und Ihr Konto sicher zu halten. |

## In-App Hints — Zahlungen

| Key | Value |
|-----|-------|
| `hint_payments_methods_title` | Zahlungsoptionen |
| `hint_payments_methods_body` | Fügen Sie eine Zahlungsmethode hinzu, um den Checkout zu beschleunigen. Ihre Zahlungsinformationen sind sicher verschlüsselt. |
| `hint_payments_security_title` | Sichere Zahlung |
| `hint_payments_security_body` | Ihre Kartendaten werden verschlüsselt und nie auf unseren Servern gespeichert. Zahlungen werden von vertrauenswürdigen Anbietern verarbeitet. |
| `hint_payments_limited_title` | Begrenzte Zahlungsoptionen |
| `hint_payments_limited_body` | Einige Zahlungsmethoden sind möglicherweise nicht in Ihrer Region verfügbar. Nachnahme ist verfügbar, wo unterstützt. |

## In-App Hints — Tracking

| Key | Value |
|-----|-------|
| `hint_tracking_explanation_title` | Live-Tracking |
| `hint_tracking_explanation_body` | Beobachten Sie die Reise Ihrer Bestellung von der Abholung bis zur Lieferung auf der Karte. |
| `hint_tracking_unavailable_title` | Tracking nicht verfügbar |
| `hint_tracking_unavailable_body` | Echtzeit-Tracking ist für diese Bestellung nicht verfügbar. Sie erhalten Statusaktualisierungen per Benachrichtigung. |
| `hint_tracking_realtime_title` | Echtzeit-Updates |
| `hint_tracking_realtime_body` | Die Karte wird automatisch aktualisiert, während Ihre Lieferung fortschreitet. |
